# **App Name**: AltMe

## Core Features:

- Photographer Profiles: Display photographer portfolios with samples, bios, genres, areas, and rates.
- Request Posting: Allow users to post photo/video requests with descriptions, dates, locations, and budgets.
- Matching System: Enable photographers to search for requests based on criteria and bid on them.
- In-App Chat: Establish dedicated chat rooms for communication after a photographer is selected.
- Content Delivery: Allow photographers to submit watermarked files via a dedicated delivery button. Non-watermarked content should be released after transaction.
- Escrow Payments: Manage payments using escrow until the client approves the delivered content.
- Recommendation Tool: Provide user recommendations by evaluating a number of factors. Generate customized prompts for requesting and reviewing photography sessions.

## Style Guidelines:

- Primary color: Soft purple (#B19CD9) for a modern and creative feel.
- Background color: Light gray (#F0F0F0) for a clean and portfolio-focused look.
- Accent color: Teal (#4DB6AC) to draw attention to calls to action and highlight text.
- Headline font: 'Poppins' sans-serif for bold titles. Body font: 'PT Sans' for readability. Note: currently only Google Fonts are supported.
- Use clean and modern icons to represent different photography genres and features.
- Visually emphasize photographer portfolios, make it easy to read request descriptions, and create a trustworthy interface.
- Employ subtle transitions and loading animations to create a polished user experience.
